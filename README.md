The M2.HAT case for Raspberry Pi5 (KUB0000036) from Kubii (https://www.kubii.com/it/case-fissaggio/4751-case-m2hat-per-raspberry-pi-5-4250236828848.html) is originally designed for use with the legacy M2.HAT, which only fit one 2230 or 2242 M.2 SSD.
This enlargement project had the scope to host the Geekworm X1004 Dual M.2 HAT (https://geekworm.com/products/x1004) which can deal with two 2280 M.2 SSDs.
If you wonder why I didn't used a Geekworm P579 case, I've tried, but even using the Armor Cooler with Cooling Fan, I experticed multiple overheat during the mdadm array sync, with that case.
